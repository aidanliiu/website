Key performance features

Can machine learning predict who wins the NBA's most prestigious individual award? This paper investigates that question using 16 seasons of game data spanning over 400 players. We trained seven models, from simple K-Nearest Neighbors to ensemble methods like Random Forest and Gradient Boosting, to predict each player's MVP vote share: a continuous value between 0 and 1 representing their proportion of first-place votes.

The results showed that Support Vector Machine and Random Forest consistently outperformed the field, closely matching actual vote shares for top candidates. KNN and AdaBoost tended to underestimate leading contenders, particularly for dominant MVP seasons like Nikola Jokic's 2021–22 campaign where his actual share was 0.875 but KNN predicted only 0.700.

Three datasets, one target

The model combines three sources to build a complete picture of each player's season: raw game performance, physical attributes, and historical voting outcomes.

Per-game metrics across 16 NBA seasons: points, assists, steals, blocks, FG%, 3P%, and minutes played.

The model in code

Below is the core training pipeline from the GitHub repository. Features are standardized before fitting, which is critical for SVM and KNN whose predictions are distance-based and scale-sensitive.

What we found

01
SVM and Random Forest are the clear leaders. Both models tracked actual vote shares within 3–5% for dominant MVP seasons, outpacing every other approach tested across both held-out seasons.
02
Boosting methods underestimate elite performance. KNN and AdaBoost systematically underpredicted vote shares for runaway winners like Jokic and Giannis, likely due to regression-to-the-mean effects in the training distribution.
03
Points and minutes are most predictive. Across all models, points per game and minutes played held the highest feature importance scores, followed by field goal percentage and assists per game.
04
Team record is the missing variable. Win percentage is a significant real-world factor in MVP voting but was absent from this dataset. Adding team performance data could meaningfully improve predictions, especially in contested races.
